Multi-Signal Scoring

Cada no de conhecimento recebe um score de 0.0 a 1.0 baseado em 5 sinais ponderados:

SignalPesoComo calcula
Tech Match0.30Interseção entre tecnologias da query e do nó. Nó sem tech = 0.5 (parcial). Nó com tech diferente = 0.0.
Domain Match0.20Interseção entre dominios. Nó "general" = 0.3 (parcial).
Severity0.20critical=1.0, high=0.8, medium=0.5, low=0.3
Reinforcement0.15count/20 (capped at 1.0). 20+ confirmacoes = score maximo.
Recency0.15≤7d=1.0, ≤30d=0.8, ≤90d=0.5, ≤365d=0.3, >365d=0.1
// Formula
score = tech_match * 0.30
      + domain_match * 0.20
      + severity * 0.20
      + reinforcement * 0.15
      + recency * 0.15

Crystallizer

Converte findings em rules. Extrai WHY do description + resolution. Extrai HOW do resolution. Detecta tecnologias do file_path.

Reinforcer

Cada vez que um finding confirma uma rule existente: reinforcement_count++ e confidence aumenta (+0.05 por confirmacao).

Promoter

L4→L3: 5 reinforcements. L3→L2: 20 reinforcements + confidence≥0.8. Cria novo no na camada superior.

Pruner

Remove: 0 reinforcements apos 60 dias. Confidence <0.05. L5 context expirado (TTL). L4 evidence >90 dias sem reinforcement.

API — Ridiculamente Facil

from engineering_brain import Brain

# Criar e popular com seeds
brain = Brain(adapter="memory")  # memory | falkordb | qdrant | full
brain.seed()  # → 117 nos carregados dos 8 YAML seeds

# Consultar (agent-transparent, context-aware)
result = brain.query(
    task_description="Write Flask server with WebSocket",
    technologies=["Flask", "SocketIO"],
    file_type=".py",
    phase="exec",
)
print(result.formatted_text)  # → 1500 chars de conhecimento relevante
print(result.total_nodes_queried)  # → ~40 nos consultados

# Adicionar regra (one-liner)
brain.add_rule(
    text="Nunca usar eval() em Python",
    why="eval() executa codigo arbitrario — injection via user input",
    how="Use ast.literal_eval() para parsear literals seguros",
    severity="critical",
    technologies=["Python"],
)

# Aprender de um finding do pipeline
brain.learn_from_finding(
    description="server.py:201 usou cors_allowed_origins=None",
    severity="high",
    resolution="Mudou para cors_allowed_origins='same-origin'",
    lesson="Flask CORS None != deny-all",
)

# Stats
print(brain.stats())
# {'layers': {'Axiom': 24, 'Principle': 19, 'Pattern': 11, 'Rule': 64, 'Finding': 1},
#  'total': 119, 'config': {'adapter': 'memory', 'budget_chars': 3000}}
